# duemate

DueMate — แอปช่วยจัดการเอกสารและวันครบกำหนด (Flutter, v0.1.0)

## DueMate v0.1.0 — Local-first MVP Foundation

**สถานะ:** ผ่านการทดสอบบน macOS desktop

### ฟีเจอร์ที่มี

- Home Dashboard พร้อม mock/in-memory data
- Add Reminder Form UI
- Mock Save กลับมาแสดงบน Home
- Reminder Detail Screen
- Search และ Filter
- Export placeholder สำหรับ PDF/CSV
- Settings/Privacy content
- UI polish เบื้องต้น
- ยังไม่ใช้ database
- ยังไม่ใช้ notification จริง
- ยังไม่ export ไฟล์จริง
- ข้อมูลที่เพิ่มเองยังเป็นข้อมูลชั่วคราวใน runtime

### ข้อจำกัด

- ปิดแอปแล้วข้อมูลที่เพิ่มเองอาจหาย
- ยังไม่ใช่เวอร์ชันสำหรับผู้ใช้จริง
- ยังไม่ build Android/iOS release
